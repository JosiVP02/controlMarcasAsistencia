# 📊 ControlMarcas - Attendance Control and Reporting System

ControlMarcas is a desktop application developed to automate attendance control, incident management, and managerial reporting processes.

The application was built using React, Tauri, and SQLite, providing a fast desktop experience for Human Resources departments to process attendance records, analyze employee attendance, manage exceptions, and generate professional reports.

---

## ✨ Features

* Employee management
* Work schedule management
* Excel attendance file import
* Automatic attendance analysis
* Late arrivals detection
* Early departures detection
* Justified absences
* Unjustified absences
* Missing clock-in/out detection
* Special schedules and exceptions
* Attendance balance calculations
* Editable attendance reports
* Managerial summary reports
* PDF report generation
* DOCX report generation
* Historical report repository
* Report download and management
* Desktop application powered by Tauri

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/JosiVP02/controlMarcasAsistencia.git
```

Navigate to the project folder:

```bash
cd controlMarcasAsistencia
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run tauri dev
```

Build the desktop application:

```bash
npm run tauri build
```
